#### Blog

> 个人博客

##### 

新建文章：
```bash
hexo new [layout] <title>
```

设置分类和标签：
```javascript
categories:
  - javascript
  - es6
tags:
  - async
```

设置文章简述：
```text
在文章中插入<!--more-->，则标记之前的部分就会成为文章的简述。
```


发布：
```bash
npm run dep
```

更换主题: 
```text
修改_config.yml里的theme即可
```
